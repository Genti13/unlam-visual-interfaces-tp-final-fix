HTML

Duración: 10 horas

Nivel: Principiante

Instructor: Gaetan Facundo

Este curso de HTML está diseñado para proporcionar una comprensión sólida de los conceptos básicos y avanzados del lenguaje de marcado de hipertexto (HTML), que es esencial para la creación y el diseño de páginas web. A lo largo del curso, los estudiantes aprenderán a estructurar documentos web utilizando etiquetas HTML, incorporar elementos multimedia como imágenes y videos, y aplicar estilos básicos para mejorar la apariencia de sus sitios web. Además, se explorarán las mejores prácticas y las últimas actualizaciones del estándar HTML5, permitiendo a los participantes crear páginas web modernas y accesibles. El curso es ideal para principiantes que desean iniciarse en el desarrollo web, así como para aquellos que buscan reforzar sus conocimientos y habilidades en HTML.

CSS

Duración: 15 horas

Nivel: Principiante

Instructor: Gaetan Facundo

Este curso de CSS (Cascading Style Sheets) está diseñado para enseñar a los estudiantes cómo dar estilo y formato a las páginas web de manera efectiva y atractiva. CSS es un lenguaje crucial para el diseño web, y este curso abarca desde los conceptos básicos hasta técnicas avanzadas de estilización. Los participantes aprenderán a utilizar selectores, propiedades y valores para controlar el diseño visual de los elementos HTML, trabajar con layout y posicionamiento, y aplicar animaciones y transiciones para mejorar la experiencia del usuario. Además, se explorarán las últimas características de CSS3, incluyendo flexbox y grid, que permiten crear diseños más complejos y responsivos. A través de proyectos prácticos, los estudiantes podrán aplicar lo aprendido y desarrollar habilidades para crear sitios web modernos y profesionales. Este curso es ideal para principiantes que quieren iniciarse en el diseño web, así como para diseñadores y desarrolladores que desean perfeccionar sus conocimientos de CSS

JS

Duración: 25 horas

Nivel: Principiante

Instructor: Gaetan Facundo

Este curso de JavaScript está diseñado para enseñar a los estudiantes cómo añadir interactividad y dinamismo a las páginas web. JavaScript es un lenguaje de programación esencial para el desarrollo web moderno, y este curso cubre desde los fundamentos hasta conceptos avanzados. Los participantes aprenderán a manipular el Document Object Model (DOM) para crear experiencias de usuario interactivas, manejar eventos, y trabajar con APIs. Además, se abordarán temas como la programación asincrónica, el uso de promesas, y las prácticas de depuración y prueba de código. A lo largo del curso, los estudiantes realizarán proyectos prácticos que les permitirán aplicar lo aprendido en situaciones del mundo real. Este curso es ideal tanto para principiantes que desean aprender a programar en JavaScript como para desarrolladores con experiencia que buscan actualizar y ampliar sus habilidades.

  • Ejemplos prácticos para mejorar tu habilidad de programación

Laravel

Duración: 10 horas

Nivel: Principiante

Instructor: Gaetan Facundo

Laravel es un framework de desarrollo de aplicaciones web con PHP, conocido por su elegancia y simplicidad. Utiliza una arquitectura MVC (Modelo-Vista-Controlador) que facilita la organización y mantenimiento del código. Laravel ofrece una gran cantidad de características útiles como enrutamiento flexible, migraciones de base de datos, plantillas Blade para la gestión de vistas, y un ORM llamado Eloquent para trabajar con la base de datos de manera intuitiva. Además, incluye características avanzadas como autenticación integrada, colas de trabajo, programación de tareas, y soporte para pruebas unitarias y de integración. Es ideal para desarrollar aplicaciones web robustas y escalables de manera eficiente.

MySQL

Duración: 15 horas

Nivel: Principiante

Instructor: Gaetan Facundo

MySQL es un sistema de gestión de bases de datos relacional de código abierto ampliamente utilizado en aplicaciones web y empresariales. Es conocido por su velocidad, confiabilidad y facilidad de uso. MySQL utiliza el lenguaje SQL (Structured Query Language) para realizar consultas y manipular datos. Ofrece características como soporte transaccional ACID (Atomicidad, Consistencia, Aislamiento, Durabilidad), almacenamiento de datos en tablas relacionales, índices para optimización de consultas, y funciones avanzadas como procedimientos almacenados y disparadores (triggers). Es compatible con una amplia gama de plataformas y se integra fácilmente con muchos lenguajes de programación y frameworks de desarrollo.

JSON

Duración: 25 horas

Nivel: Principiante

Instructor: Gaetan Facundo

JSON (JavaScript Object Notation) es un formato ligero de intercambio de datos que se ha vuelto muy popular debido a su simplicidad y facilidad de lectura y escritura tanto para humanos como para máquinas. Está basado en una estructura de pares clave-valor y es utilizado comúnmente en aplicaciones web para enviar datos entre el servidor y el cliente, así como en servicios web y APIs.

C#

Duración: 10 horas

Nivel: Principiante

Instructor: Franco Ignacio

Este curso está diseñado para aquellos que desean aprender C# desde cero. A lo largo del curso, los estudiantes se familiarizarán con la sintaxis del lenguaje, las estructuras de control, las funciones y la programación orientada a objetos. Además, se explorarán conceptos esenciales como la manipulación de datos, el manejo de excepciones y la creación de aplicaciones sencillas. Al finalizar el curso, los participantes estarán preparados para desarrollar aplicaciones en .NET y avanzar hacia áreas más especializadas como el desarrollo web, aplicaciones móviles y videojuegos con Unity.

GITHUB

Duración: 15 horas

Nivel: Principiante

Instructor: Franco Ignacio

Este curso está diseñado para principiantes que desean aprender a usar GitHub, la plataforma de control de versiones más popular. Los estudiantes aprenderán a crear y gestionar repositorios, realizar commits, crear ramas y resolver conflictos. Además, se cubrirán conceptos esenciales como pull requests, forks y colaboraciones en proyectos open source. Al finalizar el curso, los participantes tendrán las habilidades necesarias para gestionar sus proyectos de software de manera eficiente y colaborar con otros desarrolladores en proyectos de cualquier escala.

MariaDB

Duración: 25 horas

Nivel: Principiante

Instructor: Franco Ignacio

Este curso está diseñado para introducir a los participantes en el uso de MariaDB, un sistema de gestión de bases de datos relacional de código abierto y derivado de MySQL. Los estudiantes aprenderán los fundamentos de la administración de bases de datos, incluyendo la creación de bases de datos y tablas, consultas SQL básicas y avanzadas, así como la gestión de usuarios y permisos. Además, se explorarán temas como la optimización de consultas y la seguridad de bases de datos. Al finalizar el curso, los participantes estarán equipados para diseñar, administrar y mantener bases de datos utilizando MariaDB en diversos entornos de desarrollo y producción.

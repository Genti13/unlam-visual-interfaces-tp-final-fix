Camino Front-end con JS

En el camino Front-End, aprenderás a crear y diseñar sitios web interactivos desde cero. Comenzaremos con HTML, el lenguaje de marcado fundamental para estructurar el contenido web, donde dominarás etiquetas, atributos y la organización de elementos para formar páginas coherentes. Luego, avanzaremos a CSS, el lenguaje de estilos que permite embellecer y dar formato a tus páginas HTML, aprendiendo a aplicar colores, fuentes, diseños de página, y crear diseños responsivos para que tus sitios se vean bien en cualquier dispositivo.

Camino Back-end

En el camino de desarrollo Back-end, aprenderás a construir aplicaciones web robustas y eficientes utilizando estas tecnologías clave. Comenzaremos con los fundamentos de Laravel, el popular framework de PHP, para estructurar y organizar tu código de manera eficiente. Te enseñaremos cómo configurar y gestionar bases de datos MySQL, incluyendo la creación, lectura, actualización y eliminación de registros. Aprenderás a utilizar JSON para la transferencia de datos entre el Front-end y el Back-end, garantizando una comunicación fluida y eficiente.
